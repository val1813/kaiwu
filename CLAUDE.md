# kaiwu 开物 — AI Coding 增强（Plugin 已激活）

> kaiwu 已作为 Claude Code Plugin 安装。MCP 工具、斜杠命令、技能和钩子均自动可用。

## 斜杠命令
- `/kaiwu-plan` — 获取任务规划
- `/kaiwu-lessons` — 错误诊断
- `/kaiwu-record` — 记录经验
- `/kaiwu-scene` — 场景规范
- `/kaiwu-doctor` — 连接诊断
- `/kaiwu-stats` — 查看统计

## 质量规则
- 修改文件前先读取确认内容
- 局部修改失败时改用全量覆盖
- 中文 Windows: 文件操作指定 encoding='utf-8'
